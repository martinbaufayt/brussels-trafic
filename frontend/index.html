Brussels Traffic Live

Carte interactive des capteurs de trafic routier à Bruxelles, mise à jour toutes les minutes.

Source : Brussels Mobility API — … capteurs géolocalisés.

Stack :

  • Collecte automatisée via APScheduler
  • Stockage historique dans PostGIS
  • API FastAPI (GeoJSON)
  • Carte Leaflet.js

Seuls les capteurs ayant transmis des données dans la dernière minute sont affichés.
